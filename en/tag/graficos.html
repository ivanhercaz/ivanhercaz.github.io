Plotly.py, establecer por defecto el hovermode de la figura

Plotly.py is a Python library developed by Plotly to perform data analysis and data visualizations. It is very well documented, but sometimes there are some features not very clear, for example how to set by default the hovermode, the feature that allow to compare a figure with more than one trace when you have the mouse over the traces.

Este sitio web funciona gracias a Pelican, un gestor de contenido desarrollado en Python y utiliza la plantilla Buruma.
