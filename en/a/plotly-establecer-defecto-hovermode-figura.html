Plotly.py, set by default the hovermode for a figure

Plotly.py is a Python library developed by Plotly to perform data analysis and data visualizations. This tool, combined with others like pandas or NumPy, ease the creation of application to make tasks related with the data science, as well many others. This tool is very powerful and it is very well está muy bien documentada. But sometimes, however well documented they are, the configuration of some features or to perform a specific task might be more complex. This has happened to me with the hovermode.

hovermode is an attribute of layout of a figure created with Plotly.py. In this case, I create the figures with the graph_objects as follows:

#!python
import plotly.graph_objects as go

fig = go.Figure()

I am working on a project in which I am using Jupyter Notebooks on the research enviroment of Google Colab. For the data analysis that I am doing I need to create some figures visualize the data. Some of these figures are composed, in some cases, by several bar graphs. When a figure created with Plotly.py is rendered, it has a toolbar very useful. One of this tools (in the screenshot attached above) allows to compare difrentes data from the y axis for the same item from the x axis (Cartesian coordinate system. However, I need this feature activated by default due two reasons:

  1. For those who consult the analysis notebooks do not have to be activating it.
  2. For me to save time checking the data.

But even though I read the documentation over and over, I didn't get it work it. After several tries, I decided to ask in the forum of the Plotly Community [how to compare data when the mouse is hover the traces by default][plotly-py-foro-pregunta]. And the give me the answer!

Perhaps I was very dense that day, but what it is try is that activate this feature is easier than I thought and now, when I read the documentation of hovermode after this answer, I see it more clear. The solution is to set hovermode="x". Let's see an example:

:::python3
import plotly.graph_objects as go

fig = go.Figure()

fig.add_trace(
  go.Bar(
    x=["a", "b", "c"],
    y=[1, 3, 6]
  )
)

fig.add_trace(
  go.Bar(
    x=["a", "b", "c"],
    y=[2, 4, 6]
  )
)

fig.update_layout(hovermode="x")

fig.show()
